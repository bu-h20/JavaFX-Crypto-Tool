# JavaFX-Crypto-Tool
A JavaFX-based tool for encrypting and decrypting text files using Caesar Cipher, Morse Code, and DES.


Bushra’s Contribution:
Designed and developed the full JavaFX graphical user interface.
Implemented scene navigation and button actions (Encryption, Decryption, Back).
Connected the UI with all encryption/decryption algorithms.
Integrated Caesar, Morse, and DES logic into the interface.
Handled file loading, text display, and output rendering.
